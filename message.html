Message App Clone

The message clone app is run by the server. Once the server is up and running, multiple clients are able to join concurrently. The user is able to send messages to any clients in the server. They are able to send messages to a specific individual or a group message to only specific clients. They are also able to send a message to everybody in the server. This was done through threading and networking. The UI also includes a dark mode and a light mode that enables the user to switch modes depending to their liking. It also shows all the active clients that are currently in the server.
