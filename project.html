Huffman Encoding

Created a Huffman encoding app where you can encode and decode a file or a text. Similar to Huffman coding, the objective of this program is to compress a file to reduce its size without losing any detail. In the app it will first ask you to build a frequency map and choose if it's a file or a string. It will then build an encoding tree, then build an encoding map. After that, it will encode the file or the string, then you can decode it after. It will then ask you to free the memory that was allocated. Finally, you will have an option to compress and decompress the file.

Knight Escape

The objective of the puzzle is to get the white knight up to the empty square in the top right corner of the board, and to complete the puzzle with the smallest number of valid moves. It uses a standard knight moves from a traditional chess game, where a knight moves in an L-shape. This means that a knight can only move either 1) two squares in a horizontal direction, then one in a vertical direction, or 2) two squares in a vertical direction, then one in a horizontal direction. The score decreases by 5 with each valid move, and by 10 with each invalid move.

Wordle

The game's objective is pretty much the same as the actual Wordle game, it is to guess the randomly selected word within a given number of attempts. The player can select either a three or five word board. Indicators will be given if characters of the user entered word are reflected in the guessed word. If the character is in the correct position, the character will display as an uppercase value. If the character is within the random word, the character will display as a lowercase value. If you enter a character that is not in the word, an asterisk '*' will display. At the conclusion of the game, stats will be displayed.
