Custom Search Engine

This program was done by implementing an inverted index to connect the words and the URLs while making sure that the time complexity does not exceed n^2 for the efficiency of the program. This was done by first tokenizing all the words in the url given and storing it into a container like a vector. The vector is then mapped to the url. It also removes all the stop words from the vector for more efficient searching. The user is able to look up multiple words together or any words they do not want to be inlcuded in the search. Used an algorithm library for multiple words to work when looked up.
