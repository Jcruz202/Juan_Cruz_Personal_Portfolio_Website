Tax Return Data

This program reads in data using csv and json, including data from the internet. It has a csv file to work with. The csv file contains information such as the state name, the AGI stub(adjusted gross income), the zipcode, etc. Applied some math computation to differentiate tax income of different states and different AGI groups. The user will be asked to enter a state they want to evaluate, and the answers will be written in a separate file that the user will have access with. The program also has visualization like the pie chart and the vertical bar chart shown in the picture above.
